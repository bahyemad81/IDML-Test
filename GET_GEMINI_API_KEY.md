# How to Get Your Google Gemini API Key (FREE!)

Google Gemini API is **FREE** to use with generous rate limits - perfect for this translation tool!

## Step 1: Go to Google AI Studio

Visit: **https://aistudio.google.com/apikey**

## Step 2: Sign In

- Sign in with your Google account
- Accept the terms of service if prompted

## Step 3: Create API Key

1. Click **"Create API Key"** button
2. Select a Google Cloud project (or create a new one)
3. Your API key will be generated instantly

The key will look like: `AIzaSy...` (starts with "AIza")

## Step 4: Copy Your API Key

Click the **copy icon** to copy your API key to clipboard

## Step 5: Use in the Application

Paste your API key into the "Google Gemini API Key" field in the web interface

## Troubleshooting

**"API key not valid"**
- Make sure you copied the entire key (starts with AIza)
- Check that you enabled the Generative Language API in Google Cloud Console

**"Quota exceeded"**
- You've hit the free tier limit (60/min or 1500/day)
- Wait a few minutes or upgrade to paid tier
